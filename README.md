# HomeGuard Monitor

A comprehensive monitoring and alerting system for personal servers and IoT devices.

## Project Structure

```
HomeGuard-Monitor/
├── backend/          # FastAPI Python backend
├── frontend/         # React TypeScript frontend
├── agents/           # Monitoring agent scripts
├── docs/            # Documentation
├── scripts/         # Utility scripts
└── docker-compose.yml
```

## Features

- Real-time device monitoring
- Customizable alerting rules
- Multi-channel notifications (Telegram, Discord, SMS, Email)
- Time-series metrics storage
- Anomaly detection
- Interactive dashboards

## Quick Start

```bash
docker-compose up -d
```
